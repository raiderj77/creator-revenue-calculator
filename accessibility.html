Accessibility Statement

Last Updated: February 24, 2026

WCAG 2.1 AA ADA Compliant Screen Reader Friendly

Our Commitment

Creator Revenue Calculator is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

We believe that calculator tools should be accessible to all content creators, regardless of ability. Our goal is to provide an inclusive experience that works for everyone.

Accessibility Features

We have implemented the following accessibility features:

Keyboard Navigation

All calculator tools can be operated using only a keyboard. Use Tab to navigate, Enter to activate buttons, and Escape to close modals.

Screen Reader Compatibility

Proper ARIA labels, roles, and landmarks for screen readers like JAWS, NVDA, and VoiceOver.

Color Contrast

All text meets WCAG 2.1 AA contrast ratio requirements (4.5:1 for normal text, 3:1 for large text).

Responsive Design

Works on all devices from mobile phones to desktop computers with zoom support up to 200%.

Focus Indicators

Clear visual focus indicators for keyboard users with sufficient color contrast.

Alternative Text

Descriptive alt text for all meaningful images and icons.

Standards Compliance

We strive to comply with the following accessibility standards:

Web Content Accessibility Guidelines (WCAG) 2.1

Our goal is to meet Level AA of the WCAG 2.1 guidelines, which includes:

  • Perceivable: Information and user interface components must be presentable to users in ways they can perceive
  • Operable: User interface components and navigation must be operable
  • Understandable: Information and the operation of user interface must be understandable
  • Robust: Content must be robust enough that it can be interpreted reliably by a wide variety of user agents, including assistive technologies

Americans with Disabilities Act (ADA)

We aim to comply with Title III of the ADA, which requires businesses open to the public to provide equal access to individuals with disabilities.

Section 508

Our website is designed to be compatible with Section 508 of the Rehabilitation Act, which requires federal electronic and information technology to be accessible to people with disabilities.

Areas for Improvement

We recognize that accessibility is an ongoing process. Areas we are currently working to improve:

  • Enhanced voice control compatibility
  • Additional language support for screen readers
  • More detailed audio descriptions for complex calculator outputs
  • Improved error identification and recovery for form inputs

We welcome feedback on these and other accessibility features.

Technical Specifications

Accessibility of Creator Revenue Calculator relies on the following technologies:

  • HTML5 with proper semantic structure
  • WAI-ARIA for enhanced screen reader support
  • CSS for presentation and responsive design
  • JavaScript for interactive calculator functionality

These technologies are relied upon for conformance with the accessibility standards used.

Assistive Technologies and Compatibility

We test our website with the following assistive technologies:

  • Screen Readers: JAWS, NVDA, VoiceOver, TalkBack
  • Screen Magnifiers: ZoomText, Windows Magnifier
  • Speech Recognition: Dragon NaturallySpeaking, Windows Speech Recognition
  • Browser Accessibility Features: High contrast modes, text resizing, keyboard navigation

Recommended browsers for optimal accessibility:

  • Google Chrome (latest version) with accessibility extensions
  • Mozilla Firefox (latest version) with accessibility features enabled
  • Apple Safari (latest version) with VoiceOver
  • Microsoft Edge (latest version) with accessibility tools

What to Include in Your Feedback

To help us address your concern most effectively, please include:

  • The URL (web address) of the page with the issue
  • A description of the problem
  • The technology you're using (browser, screen reader, etc.)
  • Any suggestions for improvement

Formal Complaints

If you are not satisfied with our response to your accessibility feedback, you may file a formal complaint by contacting:

  • Email: complaints@creatorrevenuecalculator.com
  • Subject: Formal Accessibility Complaint

We will acknowledge receipt of your complaint within 3 business days and provide a resolution timeline.

Third-Party Content

Our website may include third-party content or links to third-party websites. We cannot guarantee the accessibility of third-party content, but we strive to work with partners who share our commitment to accessibility.

Updates to This Statement

We will review and update this accessibility statement annually or as needed to reflect changes in our website, standards, or technologies.
