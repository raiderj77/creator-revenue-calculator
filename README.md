## Setup & Deployment

### Local Development
```bash
# Clone the repository
git clone https://github.com/raiderj77/creator-revenue-calculator.git
cd creator-revenue-calculator

# Open in browser
open index.html
```

### GitHub Pages Deployment
1. Push to `main` branch
2. GitHub Pages auto-deploys
3. Custom domain: `creatorrevenuecalculator.com`
4. SSL/TLS: Automatic via GitHub Pages

### Custom Domain Setup
1. Domain purchased: `creatorrevenuecalculator.com`
2. CNAME file points to GitHub Pages
3. DNS configured at domain registrar
4. SSL certificate auto-provisioned
